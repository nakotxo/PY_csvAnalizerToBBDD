Configuración de Base de Datos

La aplicación está configurada para conectarse a una base de datos MariaDB usando las variables del archivo .env:

  • DB_HOST - Servidor de base de datos
  • DB_PORT - Puerto (por defecto 3306)
  • DB_USER - Usuario de conexión
  • DB_PASSWORD - Contraseña
  • DB_NAME - Nombre de la base de datos
Formato del Archivo
Columnas Esperadas
  • dni - Documento de identidad
  • telefono - Número de teléfono
  • email - Dirección de correo electrónico
  • Otras columnas se preservarán
Validaciones
  • DNI: 8 dígitos + letra
  • NIE: X/Y/Z + 7 dígitos + letra
  • CIF: Letra + 7 dígitos + control
  • Email: Validación RFC compliant
  • Teléfono: Prioriza móviles (6/7)
